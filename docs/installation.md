# Installation

The SDK requires Python 3.12 or newer.

With Poetry:

```console
poetry add bcic-client-api
```

Or with pip:

```console
python -m pip install bcic-client-api
```

The package is currently pre-1.0. Pin an appropriate compatible version and
read release notes before upgrading. Contributors should use the separate
[contributor setup](contributing.md).

Continue with the [quick start](quick-start.md).
